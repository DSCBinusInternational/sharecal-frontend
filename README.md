## Installation
Make sure you have node.js installed in your computer

1. Clone the repository
```sh
git clone https://github.com/DSCBinusInternational/sharecal-frontend.git
```
2. Install all the libraries
```sh
npm install
```
3. Start the dev server
```sh
npm start
```
