# sharecal-frontend
This is the greatest ShareCal Frontend react repository of all time

## Installation
Make sure you have node.js installed in your computer

1. Clone the repository
2. Install all the libraries
```sh
npm install
```
3. Start the dev server
```sh
npm start
```
